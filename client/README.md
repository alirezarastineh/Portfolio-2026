# client

This project was generated with [Analog](https://analogjs.org), the fullstack meta-framework for Angular.

## Setup

Run `pnpm install` to install the application dependencies.

## Development

Run `pnpm run dev` for a dev server. Navigate to `http://localhost:5173/`. The application automatically reloads if you change any of the source files.

## Build

Run `pnpm run build` to build the client/server project.

## Test

Run `pnpm run test` to run unit tests with [Vitest](https://vitest.dev).

## Deployment

Production deploy uses the repo-root `compose.yaml` for local Docker checks and `deploy/hetzner/docker-compose.yml` on Hetzner. See `../DEPLOYMENT.md`.
